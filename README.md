END TO END BOSTON HOUSING PRICE PREDICTION

Problem Statement:
The Boston Housing Pricing Prediction problem revolves around developing a model capable of predicting the prices of houses in Boston based on various features.

Dataset Description:
The dataset contains information on various factors influencing house prices in Boston. Common features include details like crime rates, property tax rates, the average number of rooms per dwelling, accessibility to highways, and other relevant attributes.
